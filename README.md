# Personal Portfolio Website

## Project Overview
This project is a simple personal portfolio website created using HTML.  
It showcases basic information about me, my skills, image gallery, and a contact form.

## Objectives
- Learn basic HTML structure
- Create a multi-section webpage
- Add images and navigation
- Understand form handling using JavaScript

## HTML Concepts Used
- HTML document structure
- Headings, paragraphs, lists
- Images and links
- Forms and input fields
- Sections and navigation

## Project Structure
- index.html – main webpage
- images folder – contains gallery images
- README.md – project documentation

## How to Run the Project
1. Open the project folder in VS Code
2. Right-click on index.html
3. Select **Open with Live Server**

## Output
The website opens in the browser showing navigation links, gallery items, and a contact form.
